# 股識 Stock Explorer

認識一家公司，從這裡開始。

不是股評家，是歷史學家。不喊買進賣出，只說「這家公司這些年發生了什麼事」。

## 核心理念

1. 用生活化案例說明，不是刻意教育化
2. 從全面到個人化
3. 不用冰冷數字，用實際案例推導分析
4. 不推薦股票，帶你認識股市
5. 客觀評價，只當歷史分析者

## MVP 頁面結構

### 第一頁：公司名片
- 一句話認識這家公司
- 圓餅圖：營收來源分布（白話說明每個區塊）
- 三個關鍵數字（市值、營收、毛利率）用生活化比喻
- 近期動態（重大公告/新聞白話摘要）

### 第二頁：營運健檢
- 靠什麼賺錢、賺得穩不穩、跟過去比怎樣

### 第三頁：財務體質
- 賺多少、花多少、剩多少（生活化比喻）

### 第四頁：同業比較
- 跟產業第一名差在哪（用實際案例）

### 第五頁：集團架構
- 星狀拓樸，點對點認識，逐步挖掘

## 技術架構

- 資料層：FinMind API（50+ 資料集，每日更新）
- 後端：Python + Streamlit
- 視覺化：Plotly + 自定義 CSS（PPT 風格）
- 白話解釋：LLM（限白話轉譯，不做事實解讀）
- 快取：本地檔案快取 + 失效機制

## 開發階段

### Phase 1：MVP（目前）
- 單支個股分析，5 頁完整體驗
- 手動更新資料
- 驗證：新手十秒複述正確率 > 80%

### Phase 2：進階功能
- 多股比較、集團架構完整展開、ETF 專區

### Phase 3：自動化
- 每季財報自動更新、重大事件驅動更新、追蹤訂閱通知
