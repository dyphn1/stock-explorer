## 目前側邊欄有什麼

從 `src/main.py` 第 92-156 行，目前側邊欄包含：

1. **品牌標題**：「🔍 股識」+ tagline
2. **搜尋框**：text_input，支援代號/中文名稱搜尋
3. **熱門股票**：9 支權值股按鈕（2330, 2317, 2454...）

## 目前側邊欄的結構問題

### 問題 1：搜尋框不該在側邊欄

**業界共識：** 搜尋框應該在 top-center 或 top-left 主內容區，不是側邊欄。
- Yahoo Finance：搜尋在頂部中央
- TradingView：搜尋在側邊欄頂部（但它是 icon bar + panel 系統）
- Finviz：搜尋在頂部

**建議：** 把搜尋框移到主內容區頂部（Zone A 或 Zone C 頂部），側邊欄保留導航功能。

### 問題 2：熱門股票/ETF 缺乏組織

**目前：** 平鋪 14 個按鈕，沒有分類、沒有價格資訊。

**業界做法：**
- Yahoo Finance：watchlist 內聯顯示價格 + 漲跌幅 + mini sparkline
- TradingView：watchlist 面板可切換不同清單

**建議：** 熱門股票/ETF 改為可收折的 section，每個項目顯示即時價格。

### 問題 3：側邊欄沒有分層結構

**目前：** 所有項目都在同一層級，用 `---` 分隔。

**業界做法：**
- TradingView：icon bar（頂層）→ panel（第二層）→ 內容（第三層）
- Yahoo Finance：nav items（頂層）→ watchlist groups（第二層）→ stocks（第三層）

**建議：** 建立三層結構：
1. 頂層：主要導航（Dashboard, Markets, News）
2. 第二層：Watchlists（可收折 groups）
3. 第三層：個別股票（內聯價格）
