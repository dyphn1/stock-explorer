## Document Conventions

1. **All documents are written in Markdown**, using UTF-8 encoding
2. **Bilingual naming**: Product name (Stock Explorer / Gu Shi), technical terms in English
3. **Decision records**: Any significant technical or product decision must be written to `docs/adr/`
4. **Status sync**: Update `05-roadmap.md` and `docs/state/` at the end of each Sprint
